Common styles will not be grouped.

.one { width: 100px; top: 100px; } .two { width: 100px; left: 100px; }

Common styles will be grouped by either the longest common class or all selectors together.

.one, .two { width: 100px; } .one { top: 100px; } .two { left: 100px; }

Common styles will be mapped to variables defined in the field below.

$width: 100px; .one { width: $width; top: 100px; } .two { width: $width; left: 100px; }

Common styles will be removed.

.one { top: 100px; } .two { left: 100px; }
